# BasicPosts

A full-stack, blog-style web application built with Django. This project demonstrates core backend web development concepts, including user authentication, relational databases, and complete CRUD (Create, Read, Update, Delete) operations.

## Features

* **User Authentication:** Secure user registration, login, and logout functionality.
* **CRUD Operations:** Users can create new posts, read a feed of existing posts, update their content, and delete them.
* **Authorization Security:** Access control ensures that users can only edit or delete posts that they personally authored.
* **Media Handling:** Seamless support for uploading, storing, and rendering image files alongside post text.
* **Responsive UI:** Clean, functional frontend built with Django Templates.

## Technologies Used

* **Backend:** Python, Django
* **Database:** SQLite (Django default)
* **Frontend:** HTML, CSS, Django Templates
